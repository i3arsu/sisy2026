# Images

Drop site images in this folder. Referenced from HTML/CSS as `images/<filename>`.

## Expected files

The homepage gallery (`index.html` § 04 — Pula in pictures) currently uses 5 placeholder tiles. Add files with these names to replace them:

- `pula-01.jpg` — hero / large tile (Roman arena recommended, landscape)
- `pula-02.jpg` — secondary tile
- `pula-03.jpg` — secondary tile
- `pula-04.jpg` — secondary tile
- `pula-05.jpg` — secondary tile

Suggested subjects: Pula Arena, the waterfront, the Forum, the Faculty of Economics & Tourism building, Cape Kamenjak / Istrian coast.

## Recommendations

- Format: JPEG (photos) or WebP (smaller file size)
- Width: 1600–2000px for the hero tile, 1200px for others
- Compress before committing (~150–300 KB per image is fine)
- Strip EXIF / location metadata if uploading personal photos
